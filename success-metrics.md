# Success Metrics

## Core Metrics

- Rework rate per task (number of major implementation reversals)
- Reviewer-found issue count per task
- Missed edge-case count found post-implementation
- Convention adherence score (1-5)
- Senior review effort (minutes per task)

## Baseline Capture (Before Pilot)

- Collect values from 5 comparable single-agent or manual tasks.
- Record baseline medians and ranges.

## Pilot Target (After Phase 3)

- Reduce rework rate by at least 20%
- Reduce missed edge cases by at least 25%
- Improve convention adherence by at least +1 average point
- Reduce senior review effort by at least 15%

## Data Collection Rule

Use one run log per task with strict fields to avoid subjective drift.
