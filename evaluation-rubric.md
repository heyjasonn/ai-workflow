# Evaluation Rubric (1-5)

## Axes

1. Requirement understanding
2. Design quality
3. Implementation correctness
4. Test completeness
5. Review usefulness

## Scoring Guide

- 1: unacceptable
- 2: weak
- 3: acceptable
- 4: strong
- 5: excellent

## Pass Threshold

- Average score >= 3.5
- No axis below 3 for production-facing tasks
