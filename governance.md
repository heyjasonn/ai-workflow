# Governance

## AI-Allowed Tasks

- Small/medium backend Go feature work
- Scoped bugfixes
- Simple DB migration and integration tasks

## Human-Mandatory Review

- Any security-sensitive change
- Any data migration affecting existing rows
- Any high-traffic performance-sensitive path
- Any architecture-impacting change

## AI-Restricted Tasks

- Autonomous merge to protected branches
- Autonomous production rollout
- Unapproved architecture-breaking refactors

## Approval Rule

Final output is approved by designated senior engineer or tech lead.
