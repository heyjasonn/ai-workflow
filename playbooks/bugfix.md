# Playbook: Bugfix

## Mandatory Steps

1. Researcher isolates root cause and regression scope.
2. Planner defines minimal safe fix path.
3. Implementor applies targeted change and rollback-safe behavior.
4. Tester adds regression tests.
5. Reviewer confirms risk mitigation and residual risk.

## Mandatory Outputs

- Clear bug trigger scenario.
- Regression test coverage evidence.
- Reviewer risk classification.
