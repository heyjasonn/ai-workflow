# Playbook: New Feature

## Mandatory Steps

1. Researcher handoff validated.
2. Planner design with API and DB impact.
3. Implementor code changes with explicit limitations.
4. Tester coverage report.
5. Reviewer merge readiness decision.

## Optional Steps

- Integration test step if feature touches external dependency.
- Performance review if hot path changed.

## Mandatory Outputs

- All handoff contracts valid.
- Final summary includes merge readiness and follow-up actions.
